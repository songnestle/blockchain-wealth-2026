## 通用格式示例

```csv
timestamp,asset,amount,price,fee
2025-01-01,BTC,0.5,45000,10
2025-01-02,ETH,2.0,3000,5
2025-01-03,BTC,-0.2,46000,8
2025-01-04,SOL,10.0,150,1.5
2025-01-05,ETH,1.5,3100,4.8
2025-01-06,BTC,0.3,47000,12
2025-01-07,SOL,-5.0,155,0.8

```

## 使用方法

1. 将上述内容保存为对应的CSV文件
2. 在前端界面选择对应的交易所类型
3. 上传CSV文件进行测试
